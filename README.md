## ⚠️ CRITICAL: Planning Workflow

Before writing ANY content, you MUST follow this workflow:

### Step 1: Understand the User's Need
Ask the user:
- **Who are you writing for?** (target audience)
- **What is the post goal?** (teach, challenge, story, lead magnet, etc.)
- **What's your niche/industry?**

### Step 2: Plan Which Files Are Required

**Rule:** Most posts need MORE than one skill file.

| Example Scenario | Files to Load |
|-----------------|---------------|
| SaaS founder posting about their product | `saas-tech.md` + `psychology.md` |
| SaaS founder marketing their agency services | `saas-tech.md` + `marketing-copywriting.md` + `psychology.md` |
| Coach wanting to attract clients | `coaching-personal-dev.md` + `psychology.md` |
| Coach building an agency | `coaching-personal-dev.md` + `agency-freelancing.md` + `psychology.md` |
| Freelancer in tech space | `agency-freelancing.md` + `saas-tech.md` + `psychology.md` |
| Marketer posting about marketing services | `marketing-copywriting.md` + `psychology.md` |
| Marketing agency owner | `marketing-copywriting.md` + `agency-freelancing.md` + `psychology.md` |

### Step 3: Confirm with User
Before loading files, present your plan:

```
My plan:
1. Load psychology.md (base - always required)
2. Load [niche file(s)] based on your audience
3. Combine templates from both files where relevant

Is this correct? Should I proceed?
```

**NEVER just load one file and start writing. ALWAYS plan first.**

## How to Use This System

### Understanding the Audience
The key question: **Who is this post FOR?**

- If posting FOR SaaS founders/PMs → use `saas-tech.md`
- If posting FOR marketers/copywriters → use `marketing-copywriting.md`
- If posting FOR coaches/developers → use `coaching-personal-dev.md`
- If posting FOR freelancers/agencies → use `agency-freelancing.md`

### When to Combine Files
**Real-world scenarios:**

1. **SaaS founder marketing their service:**
   - Product content = `saas-tech.md`
   - Lead gen content = `marketing-copywriting.md`
   - Combine both for best results

2. **Coach building a business:**
   - Content about coaching = `coaching-personal-dev.md`
   - Content about business/agency = `agency-freelancing.md`
   - Combine both

3. **Freelancer in tech:**
   - Content about freelance work = `agency-freelancing.md`
   - Content about tech/SaaS = `saas-tech.md`
   - Combine both

## Key Principles

1. **psychology.md is BASE** — Always load first
2. **Plan BEFORE execute** — Ask user, confirm files
3. **Combine when needed** — Most real posts need 2+ files
4. **Don't sound like marketing** — If readers feel "selling" → they scroll
5. **Specific > Generic** — Real examples, real numbers, real stories
6. **Emotion > Information** — Make them FEEL, not just understand
