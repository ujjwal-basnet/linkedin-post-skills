## What's Inside Each File

- **The Psychology of Viral Hooks** — Why emotion beats information, the 6 emotional triggers, information vs provocation distinction, authenticity filter
- Original template ratings adapted per niche
- Bad vs good hook comparisons

- Rage bait hooks for each niche

### Posts (Full Body)
- **The Psychology of Viral Posts** — Emotional arc, curiosity gap throughout, CTA psychology
- Agent instructions with niche-specific rules
- Bad vs good post comparisons
- Recreated post templates

### Lead Magnets (CTAs)
- **The Psychology of Viral Lead Magnets** — Investment signaling, FOMO stack, emotional stack

## Usage

1. Choose your niche (saas-tech, marketing-copywriting, coaching-personal-dev, agency-freelancing)
2. Load the hooks file first — write your opening
3. Load the posts file next — write your body
4. Load lead-magnets if the goal is lead generation

---

## Key Psychology Principles (In Every File)

1. **Algorithm rewards emotion, not information** — Engagement velocity + emotional intensity
2. **Information ≠ Provocation** — Make people feel something, don't just teach them
3. **The 6 Triggers** — Identity validation, status signaling, tribal belonging, productive discomfort, curiosity gap, aspiration
4. **Investment signaling** — Prove value through visible effort, don't just promise it
5. **The FOMO stack** — Social proof → transformation → exclusivity → urgency
6. **Authenticity filter** — Is this true? Would you say it in person? Are you earning the emotion? Does it serve the reader?
